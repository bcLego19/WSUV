Blackbird Project

A web dev project involving a poem.

Overview

Centered around the poem "Thirteen Ways of Looking at a Blackbird" by Wallace Stevens, this project was intended to present the poem in visually appealing ways. In addition to the poem itself, the project also includes a biography about the author. It features a tabbed style and incorporates several colors, but falls short of the desired design choices.

Technical Details

  • Technologies Used: HTML, CSS, JavaScript
  • Tools: Sublime Text
