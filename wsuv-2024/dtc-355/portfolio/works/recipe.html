Recipe Project

A web dev project about relaying a recipe.

Overview

Using typography and frontend styling, this project gives instructions for how to make pizza dough. The goal is to make an aesthetically pleasing web page that changes styles based on screen width (phone, tablet, computer). While it came out well, the coloring did not stand out in any particular way.

Technical Details

  • Technologies Used: HTML, CSS, JavaScript
  • Tools: Sublime Text
