Pytube Projects

A pet project involving the pytube library

Overview

Started mid 2024, this project involves creating a menu-based tool for interacting with the pytube library. Its current functions include searching youtube, selecting individual videos, and downlaoding audio/video files. Additional expansions to the project have yet to be realized.

Technical Details

  • Technologies Used: Python
  • Tools: Git, Github, Sublime Text, Command Prompt
  • Libraries/Frameworks: Pytube
